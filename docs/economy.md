# Economy

## Earn

Users can earn for:

- releasing original assets
- seeding files
- serving bandwidth
- mirroring allowed content
- curating shelves/playlists
- tagging/indexing data
- play-to-earn games
- tap/proof-of-engagement activities
- hosting mini-apps
- completing bounties
- compute jobs later

## Spend

Users spend on:

- boosts
- minting assets
- installing premium apps
- tipping creators
- tournament entries
- skins/items
- unlocking tools
- funding bounties
- storage priority

## Burn

Tokens burn through:

- asset mint fees
- boost fees
- anti-spam publishing
- app listing fees
- challenge penalties
- cosmetic crafting
- premium placement

## Stake

Users stake for:

- developer reputation
- storage provider trust
- curator credibility
- higher earning tiers
- pool mining
- challenge bonds
- liquidity pools

## Slash / Challenge

Bad actors can lose stake or reputation for:

- fake availability
- failed delivery
- metadata poisoning
- bot-farmed taps
- malicious app packages
- false creator claims
- spam index entries
