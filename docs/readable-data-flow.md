# Readable Data Flow

## Thesis

Free/readable text is not automatically monetizable content.

It is useful data.

The network value is:

```text
finding it
tagging it
formatting it
preserving it where allowed
connecting it
recommending it
turning it into useful indexes/datasets
```

## Sources

- fanfiction
- public-domain books
- blogs
- zines
- poetry archives
- old websites
- manuals
- recipes
- wikis
- transcripts
- personal archives
- essays
- game lore

## Modes

```text
Index-only
- Store metadata, summaries, hashes, tags, links.

Archive/mirror
- Store full text only when allowed.

Creator release
- Author intentionally publishes through chao_0x.

Dataset mode
- Structured metadata/search embeddings; full text optional.

Curation asset
- The asset is the shelf, list, graph, or recommendation route.
```

## Fanfiction stress test

Fanfiction is valuable because it is already:

- tag-heavy
- fandom-aware
- pseudonymous
- remix-friendly
- archive-oriented
- recommendation-driven
- metadata-rich

For fanfic, the asset is often the index or curation graph, not the fic itself.
