# AI Indexing Layer

AI is not a copyright god.

AI is a preparation, formatting, enrichment, and risk-scoring layer.

## Jobs

- file classification
- title/author extraction
- audio fingerprinting
- text similarity checks
- duplicate detection
- format cleanup
- summaries
- tags
- genre/mood/trope classification
- embeddings
- readable-library indexing
- fandom/ship/trope graph extraction
- lyrics/transcript extraction where allowed
- manifest generation
- storage policy recommendation
- risk mode suggestion

## Risk modes

```text
low
- user-created original, public domain, or clearly licensed

medium
- unclear source, remix, fanwork, or partial similarity

high
- likely commercial recording/text match

blocked-for-monetization
- obvious third-party commercial file attempting direct monetization
```

The AI should not guarantee legality. It should produce a risk score and route to the correct publishing mode.
