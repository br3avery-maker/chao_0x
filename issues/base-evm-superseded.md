# Base/EVM Direction Superseded

The previous Base/EVM-first economy direction is superseded.

Correct position:

```text
Native chao_0x account-chain / block-lattice ledger first.
External EVM/Base support later as bridge/export only.
```

Do not build the MVP around ERC-20, ERC-721, ERC-1155, or Base testnet contracts.

Native equivalents should be designed first:

```text
CHAO native balance
native asset-release block
native index-commitment block
native reward-claim block
native burn/stake blocks
native challenge/slash blocks
```
