# Native Block-Lattice MVP

## Goal

Build chao_0x as its own native network.

## Core rule

```text
wallet = account = account chain = node identity
```

## MVP deliverables

- native block schema
- account opening block
- signed append-only blocks
- deterministic block hashing
- send and receive block model
- release asset block
- publish index root block
- availability claim block
- conflict detection

## Non-goal

Do not build the MVP around Base/EVM contracts. External chains are bridge layers later.
