# Bridges

External chains are optional bridge/export layers, not the canonical chao_0x ledger.

Potential future bridges:

```text
EVM / Base
- wrapped CHAO
- external marketplace export
- optional liquidity access

Solana
- optional high-throughput external marketplace export

Minima
- optional sovereign-node interoperability research
```

Design rule:

```text
Native chao_0x account-chain blocks remain canonical.
Bridge receipts are secondary representations.
```
